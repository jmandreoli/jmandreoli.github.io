class PYTOOLS.ipywidgets.file_browser(path, start=None, step=50, track=True, context=(10, 5))[source]

Bases: app

An instance of this class is an app to browse the file at path, possibly while it expands. If start is None, the start position is end-of-file. If start is of type int, it denotes the exact start position in bytes. If start is of type float, it must be between 0. and 1., and the start position is set (approximatively) at that position relative to the whole file.

Parameters:

  • path (str | Path) – a path to an existing file

  • start (int | float) – index of start pointed

  • step (int) – step size in bytes

  • track (bool) – whether to track changes in the file

  • context (Tuple[int, int]) – pair of number of lines before and after to display around current position

class PYTOOLS.ipywidgets.hastrait_editor(target, default_widget_layout=None, label_layout=None)[source]

Bases: app

An instance of this class is an app to edit a traitlets structure. The construction of the widget is directed by metadata which must be associated to each editable trait in target under the key widget. This is either a callable, which produces a widget for that trait, or a dict, passed as keyword arguments to a widget guesser. Guessing is based on the trait class. Each property in default_trait_layout is applied to all the trait widget layouts which assign a None value to that property. The overall editor widget can be further customised using the following attributes:

header,footer

They can be modified freely. The header and footer widgets (VBox instances) appear immediately before and after, respectively, the trait widgets. A reset buttons is present in the toolbar by default.

label_layout

A shared Layout instance formatting the label of all the traits. Modifications of this layout therefore applies to all trait labels.

Parameters:

target (traitlets.HasTraits) – a structure with traits

class PYTOOLS.ipywidgets.animator(children=(), toolbar=(), track=None, continuity=True, **ka)[source]

Bases: app

An instance of this class is an app to control abstract animations. An animation is any callback which can be passed a frame number, enumerated by the animation widgets. The set of frame numbers is split into a sequence of contiguous intervals called tracks. Parameter track is a function which returns, for each valid frame number, the bounds of its track interval, and None for invalid frame numbers.

  • If track is an int instance, the intervals are of constant length track

  • If track is an increasing sequence of int instances, the intervals are the consecutive pairs in the sequence

  • Otherwise, track must be a function of one int input returning two int outputs

A number of other widgets are accessible in addition to main: toolbar, w_console

Parameters:

  • track (int | Sequence[int] | Callable[[int], Tuple[int, int]]) – track function

  • children – additional children widgets

  • toolbar (HBox) – additional toolbar widgets

  • continuity (bool) – if True (default), proceeds to the next track at the end of each track

  • ka – passed to the Play constructor
