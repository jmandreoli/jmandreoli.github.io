PYTOOLS.simpy — Utilities for simpy

This module provides some utilities for simpy based simulation.

Available types and functions

class PYTOOLS.simpy.RobustEnvironment(init_t=0)[source]

Bases: Environment

Instances of this class are simpy.Environment instances with method run_robust(), which extends method run to allow jumping back in the past (by rerunning since the initial time until the specified point in the past).

class PYTOOLS.simpy.SimpySimulation(*content, play_kw=None, **ka)[source]

Bases: object

An instance of this class controls the rollout of a set content of pairs where the first component is a RobustEnvironment instance and the second component is display specification. Attribute player holds a player object, created by invoking method player_factory() with keyword arguments play_kw. The player object executes the rollout in a timely fashion under user control on a display board (typically a matplotlib.Figure).

A display specification is a function which takes as input a simpy.Environment instance and a part of the display board as returned by generator method parts(), and returns a display function with no input which displays the environment on the specified part as of the time of invocation.

Parameters:

  • content (Sequence[Tuple[simpy.Environment,Callable[[simpy.Environment,Any],Callable[[],None]],...]]) – list of environments with displayers

  • play_kw (Mapping) – a dictionary of keyword arguments, passed to the player constructor (attribute factory)

  • ka – passed to method parts()

parts(fig, nrows=1, ncols=1, sharex=False, sharey=False, gridspec_kw=None, **ka)[source]

Generator of parts. This implementation assumes the board is a matplotlib grid-figure, and yields its subplots (each subplot is a part). The number of parts must be at least equal to the number of environments to which they are assigned, otherwise some environments are not processed.

Parameters:

ka – a dictionary of keyword arguments passed to the matplotlib.figure.add_subplot() method of each part (key gridlines is also allowed and denotes whether gridlines should be displayed)
